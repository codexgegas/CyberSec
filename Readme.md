# Cybersecurity Learning and Practice Platforms

  
  

This repository provides a list of platforms that can be used to learn and practice cybersecurity skills. Some of the platforms listed are free, while others require a subscription or payment. It's important to note that practicing on these platforms does not guarantee expertise or complete knowledge in the field of cybersecurity, but they can be a helpful tool to supplement other learning resources.

 

## TryHackMe

  
  

[TryHackMe](https://tryhackme.com/) is an online platform for learning and practicing cybersecurity skills through a variety of virtual environments. The platform offers different levels of challenges and scenarios, covering various cybersecurity topics such as web application security, network security, and reverse engineering. TryHackMe offers both free and paid plans.



## Hack The Box

  
  

[Hack The Box](https://www.hackthebox.eu/) is an online platform that offers a range of virtual machines to practice and learn cybersecurity skills. The platform offers different levels of challenges and scenarios, covering various cybersecurity topics such as web application security, network security, and reverse engineering. Hack The Box offers both free and paid plans.



## CTF36

  
  

[CTF365](https://ctf365.com/) is a subscription-based platform that provides a virtual environment for practicing cybersecurity skills. The platform offers different levels of challenges and scenarios, covering various cybersecurity topics such as web application security, network security, and reverse engineering.



## VulnHub

  
  

[VulnHub](https://www.vulnhub.com/) is a platform that provides virtual machines for practicing and learning cybersecurity skills. The platform offers different levels of challenges and scenarios, covering various cybersecurity topics such as web application security, network security, and reverse engineering. VulnHub offers free downloads of their virtual machines.



## PortSwigger Web Security Academy

  
  

[PortSwigger Web Security Academy](https://portswigger.net/web-security) is a free online learning platform for web application security. The platform offers a range of tutorials and challenges that cover various topics such as SQL injection, cross-site scripting (XSS), and other web application vulnerabilities.



## Google Capture the Flag

  
  

[Google Capture the Flag](https://capturetheflag.withgoogle.com/) is a free online platform for practicing and learning cybersecurity skills. The platform offers a range of challenges and scenarios that cover various cybersecurity topics such as web application security, network security, and reverse engineering.



## PentesterLab

  
  

[PentesterLab](https://pentesterlab.com/) is a platform that provides virtual machines and exercises for practicing web application security. The platform offers both free and paid plans.



## OWASP Juice Shop

  
  

[OWASP Juice Shop](https://owasp.org/www-project-juice-shop/) is a free, intentionally vulnerable web application for practicing web application security. The platform offers a range of challenges and scenarios that cover various web application vulnerabilities.



## DVWA (Damn Vulnerable Web Application)

  
  


[DVWA (Damn Vulnerable Web Application)](http://www.dvwa.co.uk/) is a free, intentionally vulnerable web application for practicing web application security. The platform offers a range of challenges and scenarios that cover various web application vulnerabilities.

## Damn Small Vulnerable Web (DSVW)

  


[Damn Small Vulnerable Web (DSVW)](https://github.com/stamparm/DSVW) is a free, intentionally vulnerable web application for practicing web application security. The platform offers a range of challenges and scenarios that cover various web application vulnerabilities.
 
  

## PicoCTF

  

  

[PicoCTF](https://picoctf.com/) is a free online platform for practicing and learning cybersecurity skills. The platform offers a range of challenges and scenarios that cover various cybersecurity topics such as web application security, network security, and reverse engineering.

## SecurityTube

  

[SecurityTube](http://www.securitytube.net/) is a platform that provides a range of courses and tutorials for learning and practicing cybersecurity skills. ## Cybrary

  

  

[Cybrary](https://www.cybrary.it/) is a platform that provides a range of courses and tutorials for learning cybersecurity skills. The platform covers various cybersecurity topics such as ethical hacking, network security, and digital forensics. Cybrary offers both free and paid plans.

  

## W3Challs

  -

  

[W3Challs](https://w3challs.com/) is a platform that provides a range of challenges and scenarios for practicing and learning cybersecurity skills. The platform covers various cybersecurity topics such as web application security, cryptography, and reverse engineering. W3Challs is free to use.

  

## CrackMe

  



[CrackMe](https://crackmes.one/) is a platform that provides a range of challenges and scenarios for practicing reverse engineering and cracking skills. The platform offers challenges for various levels of expertise, from beginners to advanced. CrackMe is free to use.
